Boss: Cloning Node

AbilityPotencyEffect
Offensive Command n/a orders some adds to use Flame Breath
Order Relay n/a orders half of the adds to use Flame Breath, followed shortly by the other half
Piercing Laser 250 telegraphed line AoE; inflicts vulnerability up (20% per stack, 1m)
Flame Breath (cloned shield dragon adds) 250 conal AoE with late telegraph; inflicts vulnerability up (20% per stack, 1m)

Notes:

  • Initial actions:
    • Offensive Command: summons 3 adds in a Y shape
      • Stand beside any of the adds
    • Offensive Command: summons 5 more adds to complete the circle
      • Stand beside the one tethered add that doesn't have another tether across from it
    • Piercing Laser
    • Order Relay - see note below for safe spots
  • Rotation (59s per cycle):
    • Piercing Laser
    • Piercing Laser
    • Offensive Command for 5 random adds
      • Stand beside a tethered add that doesn't have another tether across from it
    • Order Relay - see note below for safe spots
  • The two sets of 4 adds for Order Relay will each be a group of 3 and a solo add, separated by the solo add for the other set. To dodge the flames, start in front of the second solo add, then move to the first solo add after the first flames are done
  • A Doga demiclone will defeat the boss in about 6m with no player assistance
