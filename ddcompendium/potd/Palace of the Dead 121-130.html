Deep Palace Biast

Vulnerabilities:
AbilityPotencyEffect
Levinfang 130 instant; inflicts paralysis (30s); has to be close to use this, so ranged DPS can avoid by kiting with Sprint and/or Leg Graze
Electrify 300 telegraphed circle AoE; short cast

Notes

  • Uses Levinfang immediately followed by Electrify shortly after pull. If paralysis procs early, you won't be able to escape the AoE
  • Can use Spine Drops to remove paralysis if you're ok with not using Sustaining Potions for 1m30s

Notes

  • Can only be slowed with Arm's Length if transfigured via Pomander of Witching since it doesn't normally do melee auto-attacks
